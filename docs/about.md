# About This Portal

This GitHub Pages site is generated from repository artifacts to make contract analysis outputs transparent and easy to review.

## What is published

- Latest executive summary and scorecard for each contract.
- Full generated markdown outputs (final, analysis, issues, scorecard, executive summary).
- Immutable run history based on snapshot manifests.
- Audit trail links for run-level transparency.

## Source of truth

- Generated artifacts: `rfp-markdown/generated/`
- Immutable snapshots: `linkedin-series-archive/`
- Workflow producer: `.github/workflows/local-multistage-refinement.yml`

## Confidentiality notice

This site is configured for public GitHub Pages publication. Do not commit sensitive content that should not be publicly accessible.
